# Friendly React UI (FRUI)

A collection of vanilla react and tailwind components, fields and 
output formats written in typescript.

## Install

```bash
$ npm install frui
```

## Usage

```js
import { Alert, FieldInput, FormatMarkdown } from 'frui/react';
```
